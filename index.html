Chi-Forest: Chrono-Nutrition Vending System

Final-Year Industry Project Case Study | ENT302TC

The Challenge: The Need for Digital Differentiation

The global zero-sugar market demands that Chi-Forest maintain innovation. The existing vending machine experience is transactional, lacking the personalization and continuous user engagement needed to build strong Customer Lifetime Value (LTV) among health-conscious young professionals.

Our Solution: Chrono-Nutrition Recommendation System

We propose a smart vending machine integration leveraging circadian rhythm patterns and real-time user state (via minimal input) to offer personalized, time-sensitive beverage recommendations. This transforms the vending machine into a proactive, health-conscious retail platform.

Technical Feasibility

  • Smart Vending Integration: Mainstream IoT machines support screen updates via API; QR-code scanning and mini-program connection are mature technologies.
  • Cloud Algorithm: Recommendation logic combines time-of-day, weather, and basic user goals (e.g., "Need Energy"). Low implementation risk and high scalability.

Business Viability

  • Increase Repurchase (LTV): Personalized recommendations and loyalty points incentivize repeat purchases, directly boosting sales conversion.
  • Digital Differentiation: The innovative health experience creates a clear competitive advantage in a highly saturated zero-sugar market.
